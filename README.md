# Foodful appplication for web.

```
1. npm install
2. npm run dev
```
# Making changes, creating PR!

```
1. Clone this repo
2. Create a branch: git checkout -b <your-branch-name>
3. Make some changes, and test them.
4. Push your branch
6. Open pull request in Github.com
```

# Before making changes

```
1. git checkout main
2. git fetch
3. git pull
```
